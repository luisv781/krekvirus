# Compiling
## Requirements
- Visual Studio Code 2022
- .NET 4.7.2 or newer

## Instructions
1. Download Zip from Github (Code > Download ZIP)
2. Right-click Zip > Properties > General > Unblock
3. Extract Zip and open .sln file with Visual Studio
4. Press F5 to build
5. Exe can be found in bin > Debug
